# NBT Price Comparison Documentation

Welcome to the **Developer Documentation** for the Centralized website for price/offer comparison.

This platform empowers users to compare various financial products such as credits, insurance, exchange rates, and view bank ratings, along with tools like loan and deposit calculators.

## Getting Started

If you are a new developer on the project, please start with the [Setup Guide](setup.md) to initialize your local environment.

To understand the core underlying technologies, visit the [Architecture Overview](architecture.md).

## Project Overview

- Built with modern frontend technologies including **Vue 3**, **Inertia.js**, and **TypeScript**.
- Powered by a robust backend using **Laravel 12** and **PHP 8.2+**.
- Provides a comprehensive administrative panel using Spatie Permissions.
- Features real-time calculators and interactive maps.

Use the navigation bar and search functionality to explore the documentation.
